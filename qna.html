常見問題

一般問題

請設在開始使用任何交易單據之前一日。

執行備份程式 / 選擇需備份之公司別 / 勾選正確的SQL版本 / 選擇備份檔名 / 執行備份。

應該是每天都要做備份，並且應該使用不同儲存設備多份備份，備份的資料也要存放在公司或經營者家中，分散不同地方，以免同時毀損。

程式更新後要記得做"資料庫升級"作業。

請確認是否可正常連到伺服器 1.確認主機為開啟狀態 2.資料庫有啟動。

報表問題

只要在報表印時選擇PDF格式。

參數設定方式如下：設定 / 庫存參數設定 / 基本設定 / 數值顯示 / N。

原因是使用者在電腦的"顯示器內容"中，在"變更文字 / APP與其它項目大小"做了不同比率的設定，或是電腦的顯示器解析度不同，而造成差異。

可使用CRYSTAL REPORT工具自行定義喜歡的報表。

在報表中不要選擇"直接列印"即可。

財務問題

此問題為某些傳票未覆核，請覆核傳票後重新執行資產負債表即可。

只能建立為一張開帳傳票。

前四碼。

單張：一對一、彙總：多對一傳票、整批：期間對一傳票。

財務系統 / 財務基本資料 / 會計科目設定。
